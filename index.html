Standardize Company Name


Simple UI to demonstrate a use of XSB's Company Name Standardizer web api



Suppliers do not provide item identifying information, such as manufacturer names and part numbers, in a consistent way. This lack of standardization between vendors and data sources makes it difficult to find identical parts across the supply chain, assess pricing, regulatory conformance, and other important procurement related activities. XSB's company-name-standardizer web api standardizes manufacturer and company names so identical manufactures can be found.

The company name standardizer web api is used to standardize a company name either individually or as a batch, and to obtain batch status, errors, and results. Once company names have been standardized they can be more accurately compared. This web service can also be used to determine if a company name is a match for a cage name.
